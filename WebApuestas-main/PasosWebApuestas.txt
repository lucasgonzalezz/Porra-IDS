1. Crear cuenta render https://community.render.com/login
2. Crear cuenta neon https://console.neon.tech/
3. Descargarse los archivos del ghub: github.com/cesar04rv/WebApuestas
4. Subir los archivos a tu  ghub si los vas a modificar (repo en público)
5. Crear el proyecto de Web Services en Render, conectar con un repositorio publico y pon la url de tu repo.
6. Cambiar la regioón a una de EUW. Seleccionar el plan Free. Quedarse ahí en el paso de variable de entorno
7. Crear la base de datos PostgreSQL en Neon
Coger la Connection String y ponerla en un archivo .env de esta manera:
DATABASE_URL = 'Connection String de Neon'
8. Poner en el Render lo mismo que en el .env

Para añadir un equipo descargar el svg de https://www.footylogos.com/es/competition/laliga

guardarlo por sus 3 o 4 primeras letras
(
Barcelona-> bar.svg
Real Madird -> mad.svg
Vallladolid -> vall.svg
Valencia ->val.svg
) en la carpeta de public/Escudos



DATABASE_URL = 'Connection String de Neon'

CREATE TABLE teams (
  id SERIAL PRIMARY KEY,
  slug TEXT UNIQUE NOT NULL,  -- 'mad', 'bar', etc.
  name TEXT NOT NULL,
  active INTEGER DEFAULT 1
)
